## Installation

paquet ArchLinux:
- texlive-core
- texlive-fontsextra
- tllocalmgr ou fix pour tlmgr

paquets CTAN via tlmgr
- bbm
- upgreek (via was)
- datetime
- fmtcount
- slantsc
- titlesec
- blindtext
- changebar
- versions
- adjustbox
- xstring
- csquotes
